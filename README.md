# Sample Text Editor
  A simple text editor application developed using Python's Tkinter library. It provides essential text editing and file management          features through an easy-to-use graphical interface.

## Features
- Create and edit text files.
- Open existing text files using the file dialog.
- Save text files to the desired location and then exit using exit icon.
- User-friendly GUI built with Tkinter.

## Technologies Used
- Python
- Tkinter
- MessageBox Module
- FileDialog Module
